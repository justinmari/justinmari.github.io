game design | game development

The first spark that sent me spiraling into my interest in game design was learning how to create a simple RPG using RPGMaker (I'm a big fan of Undertale). This led to many early projects of mine, which ranged from Unity tutorials (Top-Down Space Shooter) to spontaneous bouts of wanting to code (Friend-Inspired Sidescroller Game). Using my highschool programming classes along with my earlier projects, I developed a solid foundation in the programming basics.
